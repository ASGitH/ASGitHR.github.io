dFTR - Unity!

This project, very similar to the Game Boy project of the same name (Game Boy - dFTR), the idea of this project is to transform it. Transform it from a 2D game, into a 3D one. Plus, adding more ideas that wouldn't be able to be used on the Game Boy given the limitations of the hardware. My goal for this project, complete the race while diffrent events occur, to change the dynamics of the race.
