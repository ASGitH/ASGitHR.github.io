Tronpical

Welcome to Tronpical! Go! Go! Go! Run around and try to be the last person standing! Watch out! Not only do you have to watch out for the ground below you, but for the other players around you as well. Be careful! Stand still for too long, and you'll find yourself "submerged" at the bottom of the ocean... (Weekly Game Jam - Week 213 submission)

Contributions

Player Controller
  • Logic to handle the player movement and jumping.
  • Camera handling and basing the player's movement based on camera direction.
  • Spectating controls after a certain player event.
  • Handling animation states plus passing information on when the player should change state.

    • Examples

  • Idle -> Movement -> Idle
  • Idle/Movement -> Jumping -> Idle/Movement
User Interface
  • Handling events trigger from the player.

    • Examples

  • Hosting/Joining a game
  • Canceling and going back to the previous menu

  • Taking input from the player, and displaying/updating the changes.
